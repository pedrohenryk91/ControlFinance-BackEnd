Documentação da API

POST /register

Livre para todos

Registra um usuário.

Parâmetros

[
    {
        "Email":"example@email.com",
        "Password":"string"
    },
    {
        "Email":"example@email.com",
        "Password":"string",
        "Username":"string"
    }
]

O Campo "Username" é opcional, os demais são obrigatórios

Resposta

201 Created
{
    "Description": "Usuario Criado",
    "User_Name": Username,
    "User_Email": Email
}

POST /auth/login

Livre para todos

Realiza o login do usuário

Parâmetros

{
    "Email":"example@email.com",
    "Password":"string"
}

Resposta

200 Ok
{
    "token":"jwttoken"
}

POST /register/transaction

Necessita que o usuário esteja logado

Registra uma transação.

Parâmetros

{
    "Value": number,
    "Type": "Income | Outcome"
}

Type deve ser "Income" para transações de entrada ou "Outcome" para transações de saídas

Resposta

201 Created
{
    "Description": "Registro de Transação",
    "Value": number,
    "Type": "Income | Outcome"
}

{
